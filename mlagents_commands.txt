To change environment:

1. cd C:\Users\marcd\OneDrive\Documentos\GitHub\reinforcement-learning\ml-agents-0.14.1
2. conda create -n <name environment> python=3.7
3. cd ml-agents-envs
4. pip install -e .
5. cd..
6. cd ml-agents
7. pip install -e .

-------------------------------------------------------------------------------------------

To train agents:

1. cd C:\Users\marcd\OneDrive\Documentos\GitHub\reinforcement-learning\ml-agents-0.14.1
2. conda activate mlagents0.14.1
3. mlagents-learn config/config.yaml --run-id=RollerBall-1 --train

-------------------------------------------------------------------------------------------

To use tensorflow:

1. Open a terminal or console window:
2. Navigate to the directory where the ML-Agents Toolkit is installed.
3. From the command line run: tensorboard --logdir=summaries --port=6006
4. Open a browser window and navigate to localhost:6006.
